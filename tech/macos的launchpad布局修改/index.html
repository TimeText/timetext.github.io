MacOS的Launchpad布局修改

 2022.7.7  182

Launchpad就是类似ios、ipad os上面资源库的页面一样，随着应用程序的增多，每页的app越来越慢，不想多划一下鼠标切换到下一页，所以修改每页的图标数量，一页中有几行几列。

修改行列数量命令：


# 有几列
defaults write com.apple.dock springboard-columns -int 9;

# 有几行
defaults write com.apple.dock springboard-rows -int 6;

# 重启
killall Dock

# 重新打开就能看到布局发生了改变

恢复默认排序命令：


# 类似ios、ipad os中的还原桌面命令
defaults write com.apple.dock ResetLaunchPad -bool TRUE;

# 重启
killall Dock
