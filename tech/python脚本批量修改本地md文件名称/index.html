python脚本批量修改本地md文件名称

 2022.7.6  2440

想要博客本地MarkDown文件名称按照“年月日-标题”的格式排列，可是每次创建或修改博客或标题之后都要手动修改，很是不开心，继而，写了个脚本一劳永逸。

点此展开python脚本代码
#!/usr/bin/env python
# -*- coding:utf-8 -*-

from genericpath import isdir
import os
import re
import time
from tkinter import E
from turtle import done


# 获取新文件名
def get_new_name(file):
    with open(file,'r',encoding='utf8') as md:
        line=md.readlines()
        obtitle=""
        obdate=""
        obslug=""
        for i in range(0, 11):
            if("title:" in line[i]):
                obtitle="-"+line[i][6:].replace("\n","").replace(" ","").replace("\"","")
                continue
            elif("date:" in line[i]):
                obdate=re.search(r"(\d{2}-\d{1,2}-\d{1,2})",line[i]).group(0).replace("-","")
                continue
            elif("slug:" in line[i]):
                obslug="-"+line[i][5:].replace("\n","").replace(" ","").replace("\"","")
                continue
        recname=obdate+obtitle+obslug
    return recname


# 对文件进行更名
def rename_of_file(file):   
    newname=get_new_name(file)+os.path.basename(file)[-3:]
    newfullname=os.path.dirname(file)+os.altsep+newname

# 从md文件中获取相关内容更改文件名
def sync_file_name_from_md(dir,ignore):
    newdir=dir
    if os.path.isfile(dir):
        if(os.path.basename(dir) not in ignore):
            try:
                if dir.split('.')[-1]=='md':
                    rename_of_file(dir)
            except:
                print("\n"+"!!!——"*5+"\n此处更改判断文件类型时发生错误，请处理:\n"+dir+"\n"+"!!!——"*5)
                fplog.write("\n"+"\n"+"!!!——"*5+"\n此处更改判断文件类型时发生错误，请处理:\n"+dir+"\n"+"!!!——"*5)
                os.system("pause")
    elif os.path.isdir(dir):
        for s in os.listdir(dir):
            newdir=os.path.join(dir,s)
            sync_file_name_from_md(newdir,ignore)

    dir=r'./'

模式描述
t文本模式 (默认)。
x写模式，新建一个文件，如果该文件已存在则会报错。
b二进制模式。
+打开一个文件进行更新(可读可写)。
U通用换行模式（不推荐）。
r以只读方式打开文件。文件的指针将会放在文件的开头。这是默认模式。
rb以二进制格式打开一个文件用于只读。文件指针将会放在文件的开头。这是默认模式。一般用于非文本文件如图片等。
r+打开一个文件用于读写。文件指针将会放在文件的开头。
rb+以二进制格式打开一个文件用于读写。文件指针将会放在文件的开头。一般用于非文本文件如图片等。
w打开一个文件只用于写入。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。
wb以二进制格式打开一个文件只用于写入。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。一般用于非文本文件如图片等。
w+打开一个文件用于读写。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。
wb+以二进制格式打开一个文件用于读写。如果该文件已存在则打开文件，并从开头开始编辑，即原有内容会被删除。如果该文件不存在，创建新文件。一般用于非文本文件如图片等。
a打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。也就是说，新的内容将会被写入到已有内容之后。如果该文件不存在，创建新文件进行写入。
ab以二进制格式打开一个文件用于追加。如果该文件已存在，文件指针将会放在文件的结尾。也就是说，新的内容将会被写入到已有内容之后。如果该文件不存在，创建新文件进行写入。
a+打开一个文件用于读写。如果该文件已存在，文件指针将会放在文件的结尾。文件打开时会是追加模式。如果该文件不存在，创建新文件用于读写。
