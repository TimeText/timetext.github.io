Anki是什么？

Anki是一款让你记东西更容易的软件，它比一般传统的学习方法有效，功能是非常强大的。Anki界面简洁且清晰，操作起来也很简单，设计也是很人性化的。Anki是帮助记住东西的软件，比如背单词记成语，任何人都可以通过Anki受益。

Anki官网：https://apps.ankiweb.net/

为什么搭建？

因为Anki官方的服务器部署在国外，国内用户同步起来速度很慢，尤其是对于有影音图片的牌组，搭建一台属于自己的服务器，可以快速同步牌组。

需准备什么？

1.一个云服务器，对配置没有很高的要求。

2.服务端开源项目anki-sync。

要如何搭建？

简单说就是：将服务端项目上传到服务器、解压并进入文件夹、安装所需要的文件、添加账号并设置后台运行、客户端连接

软硬件准备

本次测试的硬件及软件版本

  • 系统：Linux
  • 版本：Centos 8
  • 软件：Anki 2.1 Python3 Pip3
  • 效果：windows + 安卓 同步
  • PC客户端：anki-2.1.20-windows.exe
  • 安卓客户端：AnkiDroid-2.9.2beta4.apk

服务器端

  1. 将服务端项目下载并上传到服务器 或者 在服务器上下载并解压

  2. 安装PIP

  3. 安装pyaudio

    wget http://portaudio.com/archives/pa_stable_v190600_20161030.tgz
tar  -zxvf pa_stable_v190600_20161030.tgz
cd  portaudio
./configure  &&  make
sudo make install
vim /etc/ld.so.conf.d/portaudio.conf <<EOF
/usr/local/lib
EOF
sudo ldconfig -v
sudo pip3 install pyaudio

  4. 安装相关环境

    cd anki-bundled
pip3 install -r requirements.txt

pip3 install webob

  5. 服务器配置,在根目录下进入ankisyncd.conf,如下vi ankisyncd.conf

    [sync_app]
# change to 127.0.0.1 if you don't want the server to be accessible from the internet
host = 0.0.0.0 
# 改成127.0.0.1就是本地，0.0.0.0 接收所有连接
port = 27701 
#端口
data_root = ./collections
base_url = /sync/
base_media_url = /msync/
auth_db_path = ./auth.db
# optional, for session persistence between restarts
session_db_path = ./session.db

# optional, for overriding the default managers and wrappers
# # must inherit from ankisyncd.full_sync.FullSyncManager, e.g,
# full_sync_manager = great_stuff.postgres.PostgresFullSyncManager
# # must inherit from ankisyncd.session.SimpleSessionManager, e.g,
# session_manager = great_stuff.postgres.PostgresSessionManager
# # must inherit from ankisyncd.users.SimpleUserManager, e.g,
# user_manager = great_stuff.postgres.PostgresUserManager
# # must inherit from ankisyncd.collection.CollectionWrapper, e.g,
# collection_wrapper = great_stuff.postgres.PostgresCollectionWrapper

  6. 用户管理

    添加用户： python3 ankisyncctl.py adduser 用户名

    查看所有用户： python3 ankisyncctl.py lsuser

    删除用户：python3 ankisyncctl.py deluser 用户名

    修改密码python3 ankisyncctl.py passwd 用户名

  7. 启动

    python3 -m ankisyncd

  8. 测试

    浏览器种输入IP:port查看，网页显示Anki Sync Server则表示正常

  9. 后台运行

    安装screen：yum install screen

    新建一个screen会话：screen -S anki

    执行 python3 -m ankisyncd

    Ctrl+A,D 退出会话（退出后仍在后台运行）

    恢复会话:

    可通过screen -ls查看会话id，然后screen -x [会话id]

    或者：screen -r #恢复上一次离线的会话

    结束

    执行：exit ，会提示：[screen is terminating]，表示已经成功退出screen会话。

客户机端

windows配置

打开客户端->工具->附加组件->查看文件->打开文件夹为addons+版本号

在打开的文件夹“addons+版本号”中新建一个文件夹命名为“ankisyncd”

进入“ankisyncd”文件夹，新建文件“__init__.py”,拷贝修改以下代码到文件中：

import anki.sync, anki.hooks, aqt

addr = "http://***.***.***.***:27701/" #输入服务器IP或者本地ip
anki.sync.SYNC_BASE = "%s" + addr
def resetHostNum():
    aqt.mw.pm.profile['hostNum'] = None
anki.hooks.addHook("profileLoaded", resetHostNum)

重启Anki登录在服务器上设置的账号密码即可

安卓配置

打开客户端->设置->高级设置->自定义同步服务器->勾选使用

同步地址：修改成http://***.***.***.***:27701/

媒体同步地址：修改成http://***.***.***.***:27701/msync/

把*替换成IP
