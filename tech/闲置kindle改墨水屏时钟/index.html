最近整理博客本想删除这篇博文，但因听说下文通知的发出，遂决定留着本篇博文作为纪念。

2022年6月2日，亚马逊中国贴出公告《Kindle中国电子书店运营调整通知》，正式宣布有序退出中国电子书市场。

附件：Kindle中国电子书店运营调整通知

本篇原文如下：

  • 建一个文本文档，里面包含网页地址

    • 电脑上面新建一个文本文档，名称随意，复制粘贴以下到文档中

    • 点击进入时钟1：每分钟自动更新语句
https://XXXXX/XXXX.html
这里是你HTML单页的地址

点击进入百度：
https://www.baidu.com/

点击进入微信阅读：
http://weread.qq.com/#login

点击进入知乎热榜：
https://www.zhihu.com/hot

  • 通过邮箱发送到Kindle个人账号的文档中

    • 设置接收邮箱：打开https://www.amazon.cn/ ，登录后找到首选项-->个人文档设置-->〖发送至Kindle〗电子邮箱-->任意编辑一个自定义邮箱
    • 设置发送邮箱：打开https://www.amazon.cn/ ，登录后找到首选项-->个人文档设置-->已认可的发件人电子邮箱列表-->添加认可的电子邮箱-->输入自己的邮箱(例如QQ)
    • 发送附件到Kindle：打开邮箱-->登录后写信-->收件人为“接收邮箱”-->邮件主题内容随意-->添加附件-->选择新建的文本文档-->发送

  • 打开Kindle-->选择同步-->出现刚刚发送的附件-->选择下载-->打开-->点击相应链接即可

function update() {
  var date = new Date();
  var utc8DiffMinutes = date.getTimezoneOffset() + 480;
<!--  先使用timezoneOffset()【此方法返回的是分钟】方法获取差值，这个就得到的是格林威治时间，而北京时间相对于它是需要再加上8小时的，也就是 8 * 60 = 480分钟。-->
  date.setMinutes(date.getMinutes() + utc8DiffMinutes);

  var timeString = date.getHours() + ':' + ('0' + date.getMinutes()).slice(-2);
  var dateString = (date.getMonth() + 1) + '月' + date.getDate() + '日';
  var weekList = ['日', '一', '二', '三', '四', '五', '六'];

  var weekString = '星期' + weekList[date.getDay()];

  document.getElementById("time").innerHTML = timeString
  document.getElementById("date").innerHTML = dateString
  document.getElementById("week").innerHTML = weekString
}
