Mac 终端指令别名

由于Mac终端使用时是zsh，所有需要在zsh的配置文件中引入bash的配置文件

~/.zshrc中引入~/.bash_profile

# ~/.zshrc

# 当有 ~/.bash_profile 文件时，才加载这个文件
test -f ~/.bash_profile && source ~/.bash_profile

~/.bash_profile中输入别名设置

# ~/.bash_profile

alias blog="xxxxx/xxxx/xxxx/auto_hugo.sh"

编辑sh脚本

auto_hugo.sh所在位置应同~/.bash_profile中的一样

#!/bin/bash 

echo "hello"

别名测试

任意位置打开终端输入blog回车，输出hello，则表示设置正常。

defsave(){
	cd ${rootdir}
	git add .
	echo "===================="
	echo -n "请输入commit(回车默认为“$(date "+%y%m%d-%H%M%S")-"update"”):"
	read categories
	if [ -z  ${categories} ];then
		categories=$(date "+%y%m%d-%H%M%S")-"update"
	fi
	git commit -m ${categories}

}
defpush(){
	cd ${rootdir}
	defsave
	git push
}
defproxy(){
	cd ${rootdir}
	echo "===================="
	echo "1.查看代理"
	echo "2.设置代理"
	echo "3.取消代理"
	echo "===================="
	echo -n "请相关相关序号(回车默认=1):"
	read checkproxy
	if [ -z  ${checkproxy} ];then
		checkproxy=1
	fi
	if [ ${checkproxy} -eq 1 ];then
		git config --global --get http.proxy
		git config --global --get https.proxy
	elif [ ${checkproxy} -eq 2 ];then
		echo "===================="
		echo -n "请输入代理ip地址(回车默认为127.0.0.1):"
		read ipproxy
		if [ -z  ${ipproxy} ];then
			ipproxy=127.0.0.1
		fi
		echo "===================="
		echo -n "请输入代理端口(回车默认为1234):"
		read portproxy
		if [ -z  ${portproxy} ];then
			portproxy=1234
		fi
		git config --global http.http://github.com.proxy socks5://${ipproxy}:${portproxy}
		echo "git config --global http.http://github.com.proxy socks5://${ipproxy}:${portproxy}"
		git config --global http.https://github.com.proxy socks5://${ipproxy}:${portproxy}
		git config --global --get http.proxy
		git config --global --get https.proxy
		echo "为http和https设置代理完毕"
	elif [ ${checkproxy} -eq 3 ];then
		git config --global --unset http.proxy
		git config --global --unset https.proxy
	fi
}

延伸：使用sh脚本跳转其他目录

意图：通过tob或者其他自定义命令前往blog的主目录或其他目录内。

问题：在sh脚本内编写cd dir，执行sh脚本不能跳转。

原因：shell在执行sh脚本时候会fork一个子进程，子进程通过exec执行程序。

参考：Linux下通过Shell脚本快速进入指定目录，文章的方案是使用source执行脚本，例如source goto.sh，但和我的意图还是有所距离，一个字母都不想多输入。

解决：使用给source goto.sh加别名的方式解决

例如：

~/.bash_profile


alias tob="source /home/user/to.sh"

/home/user/to.sh

#!/bin/bash

cd /bin/

完成后使用 source ~/.bash_profile刷新。

测试：任意位置打开终端，输入tob回车后当前目录会跳转到/bin目录中。

扩展：sh数字比较

copy：bash shell数字/字符串比较

比较描述
n1 -eq n2检查n1是否与n2相等
n1 -ge n2检查n1是否大于或等于n2
n1 -gt n2检查n1是否大于n2
n1 -le n2检查n1是否小于或等于n2
n1 -lt n2检查n1是否小于n2
n1 -ne n2检查n1是否不等于n2

扩展：sh文件比较

比较描述
-d file检查file是否存在并是一个目录
-e file检查file是否存在
-f file检查file是否存在并是一个文件
-r file检查file是否存在并可读
-s fil检查file是否存在并非空
-w file检查file是否存在并可写
-x file检查file是否存在并可执行
-O file检查file是否存在并属当前用户所有
-G file检查file是否存在并且默认组与当前用户相同
file1 -nt file2检查file1是否比file2新
file1 -ot file2检查file1是否比file2旧
