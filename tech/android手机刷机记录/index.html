      • 下载地址1：androidfilehost
      • 下载地址2：sourceforge

TWRP

下载

  • 打开https://twrp.me/Devices/
  • 选择相应机型进行下载

安装TWRP

  • 方法1：TWRP app安装（需已root）

  • 方法2：REC模式TWRP安装（需已安装TWRP）

  • 方法3：快速安装

    • 下载SDK Platform Tools
    • 安装有效手机驱动
    • 在您的设备上，进入“设置”->“关于”并找到内部版本号，然后在其上点击7次以启用开发人员设置。返回并进入“开发人员选项”并启用USB调试。在计算机上，打开命令提示符并键入：
      • adb reboot bootloader //重启进入兔子模式
      • fastboot flash recovery twrp.img //刷入下载的twrp.img
      • fastboot reboot // 重新启动

  • 刷入ROM：

    • 将ROM拷贝到手机根目录
    • 进入「twrp」，点击「安装（Install）」选择刚才放在根目录的MIUI固件，刷入。
    • 三清：在「清除（Wipe）」里面选择Dalvik Cache 、 Cache、Data，完毕后重启手机。

OPPO R9m获取ROOT和安装第三方REC

  • 网上下载低版本的ROM：R9m_11_A.13_OTA_013_all_201603140221.zip
  • 进入REC刷入低版本，重新开机
  • 下载KingROOT进行ROOT权限获取，NewKingrootV4.85_C139_B255_office_release_2016_03_29_105003.apk
  • 下载安装小欧工程师APP，授权root权限，刷入第三方REC即可
