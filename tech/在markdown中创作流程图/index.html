mermaid

Mermaid 是一个用于画流程图、状态图、时序图、甘特图的库，使用 JS 进行本地渲染，广泛集成于许多 Markdown 编辑器中。

graph有向图

graph LR #图 方向

  • TB - top bottom
  • BT - bottom top
  • RL - right left
  • LR - left right
  • TD - same as TB

形状

  • 方框[]
  • 圆角框()
  • 圆圈(())
  • 半方框A>B]
  • 正方形框{}

连线

  • 箭头直线 -->
  • 无箭头直线---
  • 注释：A--注释--B或者A-- |注释|B
  • 虚线箭头- . ->
  • 无箭头虚线-.-
  • 注释：-.注释.->
  • 加粗箭头==>
  • 加粗注释箭头==注释==>

flow标准流程图

横向

st=>start: 开始框

op=>operation: 处理框

cond=>condition: 判断框(是或否?)

sub1=>subroutine: 子流程

io=>inputoutput: 输入输出框

e=>end: 结束框

st(right)->op(right)->cond

cond(yes)->io(bottom)->e

cond(no)->sub1(right)->op

纵向

st=>start: 开始框

op=>operation: 处理框

cond=>condition: 判断框(是或否?)

sub1=>subroutine: 子流程

io=>inputoutput: 输入输出框

e=>end: 结束框



st->op->cond

cond(yes)->io->e

cond(no)->sub1(right)->op
