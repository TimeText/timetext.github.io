绝对路径：从根目录算起的路径叫做绝对路径，Windows 的根目录是指定的盘符，mac OS 和Linux 是/

相对路径：从当前操作 html 的文档所在目录算起的路径叫做相对路径，./表示当前文件所在目录下，../表示上级目录下

一般都会使用相对路径，绝对路径的操作在其它电脑上打开会有可能出现资源文件找不到的问题

链接标签：<a> #### </a> 可以在网页上定义一个链接地址，href属性 定义跳转的地址，title属性 定义鼠标悬停时候弹出的文字提示框，target属性 定义链接窗口打开的位置,target="_self"在当前页面打开，target="_blank"在新页面打开。

<table> table标签常用属性：</table>
border:定义表格的边框，设置值是数值；
cellpadding：定义单元格内容与边框的距离，设置值是数值；
cellspacing：定义单元格与单元格之间的距离，设置值是数值；
align：设置整体表格相对于浏览器窗口的水平对齐方式，设置值包括：left，center，right

单元格标签常用属性：</table>
align:设置单元格中内容的水平对齐方式，设置值有：left,center,right
valign:设置单元格中内容的垂直对齐方式，设置值有：top,middle,botton
colspan:设置单元格水平合并,设置值是数值
rowspan:设置单元格垂直合并，设置值是数值

表单标签：表单用于搜集不同类型的用户输入，作用就是可以把用户输入数据一起提交到web服务器。
<form>定义整体表单区域
<label>表示表单元素的文字标注标签，定义文字标注
<input>表示表单元素的用户输入标签，定义不同类型的用户输入数据方式
type 属性：

  • type="text" 定义单行文本输入框
  • type="password" 定义密码输入框
  • type="radio" 定义单选框
  • type="checkbox" 定义复选框
  • type="file" 定义上传文件
  • type="submit" 定义提交按钮
  • type="reset" 定义重置按钮
  • type="button" 定义一个普通按钮
  • type="image" 定义图片作为提交按钮，用src属性定义图片地址,一般会导致提交两次，不建议使用
  • type="hidden" 定义一个隐藏的表单域，用来存储值
<textarea>定义多行文本输入框
<select>定义下拉表单元素
<option> 标签与 <select>标签配合，定义下拉表单元素中的选项
