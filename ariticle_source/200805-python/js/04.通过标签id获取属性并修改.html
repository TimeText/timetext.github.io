获取标签元素

  • 使用内置对象document上的getElementById方法获取页面上设置了id属性的标签元素
  • 页面代码从上往下加载执行，所有JS获取的元素要在JS代码上方，解决方法有两个
    • 将JS代码放在最下边
    • 设置页面加载完成执行的函数[window.onload = function()]，在执行函数内获取标签元素
      • onload是页面所有元素加载完成的事件，给onload设置函数时，当事件触发就会执行设置的函数。

属性的操作

  • 获取页面标签元素，然后对标签元素的属性进行操作
  • 属性的读取和设置
  • 属性名在js中的写法
    • html属性和js属性大多一样，但是"class"属性写成"className"
    • "style"属性里面的属性，有斜杠的改成驼峰式，比如"font-size"改成"style.fontSize"
